# ember-rails-demo

Run Backend Rails API server localhost:3000
```sh
$ cd backend
$ bundle install
$ rake db:migrate
$ rake db:seed
$ rails s
```

Run Frontend EmberJS Server localhost:4200
```sh
$ cd frontend
$ npm install
$ ember server
```
go to browser http://localhost:4200
